# Aigentur Platform

Monorepo for the Aigentur content platform.

- `studio/` – Sanity Studio for editing content
- `frontend/` – Next.js app for rendering public blogs (e.g. blog.clientdomain.com)

## Getting started

```bash
npm install
npm run dev:studio   # run Sanity Studio on http://localhost:3333
npm run dev:frontend # run Next.js frontend on http://localhost:3000
```

You'll need Node.js 18+ and npm 9+ installed.
